## Status: NOT PART OF THE VALIDATED FORENSIC METHOD

The Audit Verification Tool (AVT) is **explicitly excluded** from the validated forensic processing pipeline of Project Aletheia.

### Purpose
The AVT exists solely to support:
- Independent technical review
- Defence disclosure obligations (CPIA 1996)
- Post-hoc verification of report, ledger, and method integrity

### Regulatory Position
- **Not** used during forensic analysis
- **Not** relied upon to generate evidential findings
- **Not** part of the ISO/IEC 17025 validated method set
- **Not** required for prosecution use

### Compliance Clarification
Under UK FSR Statutory Code v2:
- AVT supports transparency and auditability
- AVT does **not** influence evidential conclusions
- Failure or absence of AVT has no impact on forensic results

### Accreditation Boundary
The validated forensic baseline is frozen at:

> Project Aletheia v1.0.0

Any code under `tools/audit-verify/` exists **outside** that boundary.

### Intended Users
- Defence experts
- Independent auditors
- Court-appointed technical reviewers

### Explicit Exclusion
The AVT **MUST NOT** be referenced as a forensic method in expert testimony.
